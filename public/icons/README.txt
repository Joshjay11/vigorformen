# Icons for VigorForMen PWA

This directory should contain the following icon files for Progressive Web App (PWA) support:

1. icon-192x192.png (192x192 pixels)
2. icon-512x512.png (512x512 pixels)

These icons are referenced in the manifest.json file and are required for PWA installation.

You can generate these icons using tools like:
- https://favicon.io/
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/

Make sure the icons have a transparent background and represent your brand effectively.
